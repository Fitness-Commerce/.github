## projects 사용법 및 이슈 컨벤션 정리
https://zero-study-2.notion.site/projects-issue-649189daf1a24261ae8dcd71ea2a469d?pvs=4

| 이슈 유형 | 설명 |
| --------- | ---- |
| front | 프런트 업무 |
| back | 백엔드 업무 |
| Fixes | 이슈 수정중 (아직 해결되지 않은 경우) |
| Resolves | 이슈를 해결했을 때 사용 |
| Ref | 참고할 이슈가 있을 때 사용 |
| Related to | 해당 커밋에 관련된 이슈번호 (아직 해결되지 않은 경우) |
